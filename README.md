IOIO_COLOR_RECOGNITION_SENSOR
=============================

IOIO COLOR RECOGNITION SENSOR (TSC230 COLOR SENSOR)  
(Control the hardware by Android application and IOIO board)

## PIN MAP
###### TSC230 Color Sensor & IOIO  
  
S0 = PIN 1  
S1 = PIN 2  
  
S2 = PIN 4  
S3 = PIN 5  
  
OUT = PIN 35  
  
OE = PIN GND  
  
VCC = PIN +5V  
GND = PIN GND
